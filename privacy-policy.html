Privacy Policy for ImageIntact

Effective Date: August 14, 2025

Your Privacy Matters: ImageIntact is designed with privacy at its core. We don't collect, store, or transmit any personal data or information about your files.

Overview

ImageIntact is a local backup application for macOS that helps you create verified backups of your image and video files. This privacy policy explains how the app handles your data and respects your privacy.

Information We Don't Collect

ImageIntact does NOT collect, store, or transmit:

  • Personal information
  • File contents or metadata
  • Usage statistics or analytics
  • Location data
  • Device identifiers
  • Crash reports (unless explicitly shared by you)

How ImageIntact Works

Local Processing Only: All file operations happen entirely on your Mac. Your files never leave your computer.

File Access: ImageIntact only accesses folders and files that you explicitly select using the standard macOS file picker. The app uses macOS security-scoped bookmarks to remember your selected folders between sessions, stored only in your local preferences.

No Cloud Services: ImageIntact does not connect to any cloud services or external servers for its core functionality.

Network Connections

ImageIntact may connect to the internet only for:

  • Update Checks: Optional checks for new versions from GitHub releases (no personal data sent)
  • Network Drives: Accessing network-attached storage that you explicitly select as a backup destination

These connections can be disabled or are only made when you initiate them.

Data Storage

ImageIntact stores only:

  • Your preferences (selected folders, app settings) in standard macOS preference files
  • Security-scoped bookmarks for folders you've granted access to

All data is stored locally on your Mac and is never transmitted elsewhere.

Third-Party Services

ImageIntact does not integrate with or send data to any third-party services, analytics platforms, or advertising networks.

Data Security

Your file security is maintained through:

  • Using standard macOS security features and permissions
  • Never modifying your original files (read-only access)
  • Performing SHA-256 checksums locally for verification
  • Respecting macOS sandboxing and security policies

Children's Privacy

ImageIntact is a general audience application that does not collect any personal information from anyone, including children.

Changes to This Policy

Any updates to this privacy policy will be posted on this page with an updated effective date. As ImageIntact doesn't collect user data, we cannot notify you directly of changes.

Open Source

ImageIntact's source code is available on GitHub, allowing you to verify our privacy practices.

Contact

If you have questions about this privacy policy or ImageIntact's privacy practices, you can:

  • Open an issue on GitHub
  • Contact the developer through the Mac App Store
